# 读出来

使用微软 Edge TTS 语音朗读 Markdown 和 PDF 文档的 VS Code 扩展。

语音由微软 Edge 在线 TTS 服务提供（晓晓 / zh-CN-XiaoxiaoNeural），自然流畅，免费无需 API Key。

## 功能

- 朗读 Markdown 和 PDF 文档
- 播放 / 暂停 / 停止 / 上一段 / 下一段
- 进度条点击跳转到任意段落
- 调速 0.25x ~ 3.0x（即时生效，无需重新生成音频）
- 当前朗读文本实时显示

## 使用方法

1. 打开一个 Markdown 或 PDF 文件
2. 点击编辑器右上角的 ▶ 按钮，或按 `Ctrl+Alt+R`，或运行命令 `读出来: 开始朗读`
3. 在弹出的面板中控制播放

## 要求

- 需要网络连接（使用微软 Edge TTS 在线服务生成语音）
